Ejemplo Básico de Red Neuronal

Construyendo un Perceptrón Multicapa (MLP) desde cero con TensorFlow y Keras.

Llevando la Teoría a la Práctica

A continuación, vamos a implementar nuestra primera red neuronal en Python. Para ello, haremos uso de las librerías TensorFlow y Keras.

Haremos un ejemplo sencillo con un dataset que ya conocemos en Machine Learning: el conjunto de datos Iris. Es un problema de clasificación multiclase sencillo de resolver, que usaremos para explicar poco a poco cómo montamos desde cero una arquitectura de Deep Learning capaz de enfrentarse a este escenario.

Concretamente, vamos a implementar un Perceptrón Multicapa (MLP), que es una arquitectura de red neuronal relativamente simple, que consta solo de elementos que ya hemos explicado a lo largo de este tema.

⚠️ Atención: Entorno de Ejecución

Para el entrenamiento de Redes Neuronales de manera eficiente, es imprescindible contar con una GPU correctamente configurada. En caso de no tener un entorno local de este tipo (o usar un ordenador estándar), es altamente recomendable ejecutar el código en plataformas cloud que sí estén preparadas gratuitamente para este tipo de usos, como puede ser el caso de Google Colab.

1. Importación de Librerías

Primero, cargaremos todas las librerías que vamos a usar en nuestro script, tanto de manipulación de datos como de deep learning.

1_imports.py 
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from tensorflow.keras.utils import to_categorical

2. Carga y Preparación de los Datos

A continuación, cargaremos y prepararemos los datos con técnicas que ya hemos visto. Por un lado, escalamos nuestros datos estructurados usando StandardScaler. Luego, como nuestra salida es multiclase (3 especies de flores), la codificaremos con un one-hot encoding (to_categorical), de manera que cada neurona de la última capa mapeará la posibilidad de pertenencia a cada una de las clases de salida.

2_data_preparation.py 
# Cargar y extraer variables del dataset
iris = load_iris()
X = iris.data
y = iris.target

# Escalado de las variables numéricas (para ayudar a la convergencia de la red)
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Codificación one-hot de las etiquetas de la variable objetivo
y_encoded = to_categorical(y)

# División en entrenamiento y prueba (80% / 20%)
X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y_encoded, test_size=0.2, random_state=42
)

3. Construcción de la Red Neuronal (Topología)

Ahora vamos a construir la red. Crearemos una red neuronal secuencial (Sequential, las capas se apilan ordenadamente) de tres capas densas (cada neurona de una capa está conectada a todas las neuronas de la capa siguiente).

  • 1ª Capa (Entrada + Oculta): Tendrá 8 neuronas y un tamaño de entrada de 4, ya que nuestros datos tabulares tienen 4 características. Utiliza una función de activación "ReLU", que introduce no linealidad ("apagando" valores negativos) para aprender patrones complejos.
  • 2ª Capa (Oculta): Tiene 6 neuronas, toma de entrada la salida de la primera y vuelve a usar "ReLU".
  • 3ª Capa (Salida): Genera el output (probabilidad). Tiene 3 neuronas (tantas como clases). Como función de activación, cambiamos a "Softmax", la cual toma los valores resultantes y los transforma en una distribución de probabilidad (conjunto de 3 números que sumarán exactamente 1).

3_model_construction.py 
# Construcción de la arquitectura del modelo
model = Sequential([
    Dense(8, input_shape=(4,), activation='relu'),
    Dense(6, activation='relu'),
    Dense(3, activation='softmax') # 3 clases de salida en Softmax
])

4. Compilación del Modelo

Una vez definido el modelo, tenemos que "compilarlo". Aquí especificamos el optimizador, la función de pérdida y las métricas:

  • Optimizador: Ajusta los pesos internos. Usamos Adam por ser el más popular y efectivo.
  • Función de Pérdida (Loss): Para minimizar el error, al ser un problema multiclase utilizamos categorical_crossentropy.
  • Métricas: Para entender qué tan bien va (monitorearlo), usamos accuracy (porcentaje de aciertos), que es fácil de interpretar por humanos.
4_model_compile.py 
# Compilación del modelo definiendo las piezas clave para el aprendizaje
model.compile(
    optimizer='adam',
    loss='categorical_crossentropy',
    metrics=['accuracy']
)

5. Entrenamiento del Modelo

Para instruir al modelo, surgen nuevos parámetros respecto al Machine Learning clásico:

🔄 Epochs (Épocas)

Definen cuántas veces el modelo repasará todo el dataset. Si ajustamos 100, la red revisará y refinará pesos 100 veces. ¿Poco? Underfitting. ¿Mucho? Overfitting.

📦 Batch Size

El número de muestras que procesa a la vez. No mira todo de golpe. Si lo fijamos a 8, analizará de 8 en 8, evaluando el error y modificando pesos cada 8 datos.

🛡️ Validation Split

Le indica al modelo que reserve un trozo de los datos (ej: 0.1 = 10%) para verificar la pérdida de validación en cada época y alertarnos de posible overfitting.

5_model_training.py 
# Entrenamiento iterativo del modelo
history = model.fit(
    X_train,
    y_train,
    epochs=100,
    batch_size=8,
    verbose=0, # Evita que imprima todo el proceso interno en consola
    validation_split=0.1
)

6. Evaluación y Predicciones

En este punto, ya tenemos entrenada nuestra red neuronal y podemos validar su rendimiento sobre los datos que apartamos (Test) o directamente, usarla para realizar predicciones probabilísticas de nuevas flores en producción.

6_model_evaluation.py 
# Evaluación cuantitativa en las métricas dictadas anteriormente
loss, accuracy = model.evaluate(X_test, y_test, verbose=0)
print(f"Accuracy del modelo en el conjunto de prueba: {accuracy:.2f}")

# Predicción sobre nuevas observaciones (sacando únicamente 5 datos como demo)
predicciones = model.predict(X_test[:5])
print("\nEjemplo de predicciones (distribución de probabilidades por clase):\n", predicciones)

Código Completo del Script

En resumen, hemos implementado desde cero una red neuronal capaz de resolver un problema de clasificación multiclase, con una arquitectura adecuada para el problema. A continuación tienes todo el código estructurado listo para ejecutar:

keras_mlp_iris_completo.py 
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from tensorflow.keras.utils import to_categorical

# 1. Cargar y preparar los datos
iris = load_iris()
X = iris.data
y = iris.target

# Escalado de variables numéricas
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Codificación one-hot de las etiquetas
y_encoded = to_categorical(y)

# División en entrenamiento y prueba
X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y_encoded, test_size=0.2, random_state=42
)

# 2. Construcción del modelo (Arquitectura MLP)
model = Sequential([
    Dense(8, input_shape=(4,), activation='relu'),
    Dense(6, activation='relu'),
    Dense(3, activation='softmax') # 3 clases de salida (Probabilidad)
])

# 3. Compilación del modelo
model.compile(
    optimizer='adam', 
    loss='categorical_crossentropy', 
    metrics=['accuracy']
)

# 4. Entrenamiento del modelo
history = model.fit(
    X_train,
    y_train,
    epochs=100,
    batch_size=8,
    verbose=0,
    validation_split=0.1
)

# 5. Evaluación del modelo y extracción de resultados
loss, accuracy = model.evaluate(X_test, y_test, verbose=0)
print(f"Accuracy del modelo en el conjunto de prueba: {accuracy:.2f}")

# 6. Predicción sobre observaciones del lote de test
predicciones = model.predict(X_test[:5])
print("\nEjemplo de predicciones (probabilidades):\n", predicciones)
