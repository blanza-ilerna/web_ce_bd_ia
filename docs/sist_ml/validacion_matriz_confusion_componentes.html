6.4.1 Componentes de la Matriz de Confusión

Desglosando el rendimiento: los cuatro pilares de la clasificación.

La Cuadrícula de la Verdad

La matriz de confusión organiza las predicciones del modelo frente a los valores reales en cuatro elementos críticos que resumen todos los resultados posibles.

Cada componente representa una combinación específica entre lo que el modelo "dijo" que pasaría y lo que "realmente" ocurrió. Entender estas combinaciones es la base para diagnosticar cualquier sistema de clasificación.

✅ Verdadero Positivo (TP - True Positive)

El modelo predice la clase positiva y el evento efectivamente ocurre. Son los aciertos en la detección del caso de interés.

Ejemplo: Un sistema de detección de enfermedades identifica correctamente a un paciente enfermo.

⚠️ Falsos Positivos: Error Tipo I (FP)

El modelo predice positivo, pero la realidad es negativa. Es una falsa alarma. La métrica que lo controla es la Precision.

Ejemplo: Un filtro de spam bloquea un correo legítimo, o un detector oncológico da positivo en un paciente sano.

🔘 Verdadero Negativo (TN - True Negative)

El modelo predice negativo y efectivamente el evento no ocurre. Refleja la capacidad para descartar correctamente los casos normales.

Ejemplo: Un test de embarazo confirma correctamente que una persona no está embarazada.

❌ Falsos Negativos: Error Tipo II (FN)

El modelo predice negativo, pero la realidad es positiva. Es una omisión o fallo en la detección. La métrica que lo controla es el Recall.

Ejemplo: Un detector de fraude deja pasar una transacción robada, o un test médico no detecta una enfermedad real.

Interpretación del Comportamiento

La distribución de estos cuatro valores define la "personalidad" del modelo:

Modelo Conservador

Tiene pocos falsos positivos pero muchos falsos negativos. Solo predice positivo cuando está muy seguro, arriesgándose a perder casos reales.

Modelo Agresivo

Maximiza la detección (pocos falsos negativos) pero a costa de generar muchas falsas alarmas (muchos falsos positivos).

Atención

Para no confundirte, recuerda esta regla nemotécnica: Positivo/Negativo indica lo que el modelo predice. Verdadero/Falso indica si el modelo ha acertado o no respecto a la realidad.
