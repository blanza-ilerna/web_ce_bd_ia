Escalado Estándar (Z-Score)

Estandarizando la distribución de las variables numéricas.

¿Qué es el StandardScaler?

El Escalado Estándar (también conocido como estandarización o Z-score normalization) transforma los datos de manera que la distribución resultante tenga una media de 0 y una desviación estándar de 1.

A diferencia del Min-Max, este método no comprime los datos en un rango estricto, sino que los centra respecto a su promedio estadístico.

X' =
X - μ σ

μ = 0

Media aritmética

σ = 1

Desviación estándar

Diferencias clave con Min-Max

Manejo de Outliers

El escalado estándar es menos sensible a los valores atípicos. Como no depende de un máximo y mínimo absolutos para definir el rango, los outliers conservan su distancia relativa sin "aplastar" al resto de los datos.

Centrado en Cero

Al tener media cero, los valores se distribuyen tanto en el lado positivo como en el negativo, lo cual es fundamental para el rendimiento de muchos optimizadores matemáticos.

Implementación en Python

Utilizamos el estimador StandardScaler de Sklearn para realizar esta transformación de forma eficiente.

standard_scaling.py 
from sklearn.preprocessing import StandardScaler

# Inicializamos el estandarizador
scaler = StandardScaler()

# Seleccionamos las columnas a procesar
cols = ['edad', 'altura', 'peso']

# Aplicamos la transformación
# Los datos ahora tendrán media 0 y desviación 1
df[cols] = scaler.fit_transform(df[cols])
