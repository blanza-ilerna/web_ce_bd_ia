√Årboles de Decisi√≥n

Mapas l√≥gicos para la toma de decisiones automatizada

¬øC√≥mo razona un √°rbol?

Los √°rboles de decisi√≥n son modelos jer√°rquicos que dividen el espacio de datos en regiones m√°s simples mediante reglas condicionales. Su funcionamiento es similar a un juego de preguntas: "¬øEs el color rojo?", "¬øEs el precio mayor a 50‚Ç¨?".

En cada nodo del √°rbol se realiza una partici√≥n seg√∫n el valor de una variable, descendiendo por las ramas hasta llegar a las hojas, donde se asigna la predicci√≥n final.

üß† Interpretabilidad

Permiten representar el razonamiento del modelo de forma expl√≠cita y visual.

‚ö° Flexibilidad

Manejan tanto variables num√©ricas como categ√≥ricas sin necesidad de mucho preprocesamiento.

‚ö†Ô∏è Sobreajuste

Tienden a memorizar los datos si no se limita su profundidad (max_depth).

üíª Ejemplo 1: Clasificaci√≥n (Iris)

Entrenamos un √°rbol para identificar especies de flores bas√°ndonos en sus medidas. 

from sklearn.tree import DecisionTreeClassifier, plot_tree
from sklearn.model_selection import train_test_split
from sklearn.datasets import load_iris
import matplotlib.pyplot as plt

# Cargar datos
iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(iris.data, iris.target, test_size=0.2, random_state=42)

# Crear modelo limitando la profundidad para evitar sobreajuste
clf = DecisionTreeClassifier(max_depth=3, random_state=42)
clf.fit(X_train, y_train)

# Precisi√≥n
print(f"Precisi√≥n: {clf.score(X_test, y_test):.4f}")

# Visualizar la estructura
plt.figure(figsize=(12,8))
plot_tree(clf, feature_names=iris.feature_names, class_names=iris.target_names, filled=True)
plt.show()

üìâ Ejemplo 2: Regresi√≥n (Diabetes)

Los √°rboles tambi√©n pueden predecir valores continuos agrupando instancias similares en las hojas. 

from sklearn.tree import DecisionTreeRegressor
from sklearn.datasets import load_diabetes
from sklearn.metrics import r2_score

# Cargar datos
diabetes = load_diabetes()
X_train, X_test, y_train, y_test = train_test_split(diabetes.data, diabetes.target, test_size=0.2, random_state=42)

# Entrenar regresor

regressor.fit(X_train, y_train)

# Evaluaci√≥n
y_pred = regressor.predict(X_test)
print(f"Puntaje R¬≤: {r2_score(y_test, y_pred):.4f}")
