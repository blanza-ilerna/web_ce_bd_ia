Variables Cuantitativas

La esencia numérica que alimenta los algoritmos de IA.

¿Qué son las Variables Cuantitativas?

Son aquellas cuyos valores representan una magnitud numérica intrínseca. A diferencia de las etiquetas textuales, estas reflejan cantidades que podemos medir o contar de forma objetiva.

Procesamiento Nativo

Al ser numéricas por naturaleza, permiten realizar operaciones aritméticas significativas (sumas, promedios, desviaciones). Son el lenguaje "natural" de la gran mayoría de modelos matemáticos y estadísticos.

🧮

Clasificación de los Datos Numericos

Dependiendo de cómo se originen los datos, distinguimos dos categorías fundamentales:

1. Variables Discretas

Surgen de un proceso de conteo. Toman valores aislados, generalmente enteros, donde existe un "salto" claro entre una cifra y la siguiente.

  • 👨‍👩‍👧‍👦 Número de hijos (0, 1, 2...)
  • 📦 Stock de inventario
  • 🖱️ Visitas a una web
  • 🎂 Edad (años cumplidos)

2. Variables Continuas

Son el resultado de un proceso de medición. Pueden adoptar infinitos valores dentro de un rango determinado, limitados solo por la precisión del instrumento.

  • 📏 Altura (e.g., 1.753m)
  • 🌡️ Temperatura (e.g., 21.45°C)
  • ⚖️ Peso (e.g., 68.7 kg)
  • ⏱️ Tiempo de respuesta

Precauciones Fundamentales

No todos los números en un dataset son variables cuantitativas, y no todas las jerarquías son simples etiquetas. La confusión entre estos tipos es una fuente común de errores en el análisis de datos.

⚠️ ¡Atención! Riesgos de una mala interpretación

Pérdida de Jerarquía: Si tratamos una variable ordinal como si fuera una simple etiqueta nominal, estamos desperdiciando información valiosa sobre el orden y la progresión de los datos.

Cálculos Absurdos: Intentar calcular la "media" de una variable nominal (aunque use números, como un Código Postal o un ID de usuario) produce resultados estadísticamente nulos y carentes de sentido profesional.
