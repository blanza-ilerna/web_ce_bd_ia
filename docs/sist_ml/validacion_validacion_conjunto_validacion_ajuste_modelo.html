6.3.1 Conjunto de Validación y Ajuste

Decisiones maestras: el rol de la validación en la optimización.

El Rol del Dataset de Validación

El conjunto de validación es un subconjunto de datos separado del entrenamiento que se utiliza para ajustar el modelo en un sentido amplio.

Su función no es entrenar parámetros internos (como los pesos), sino permitirnos tomar decisiones sobre su configuración: hiperparámetros, algoritmos, arquitecturas o transformaciones. A diferencia de los parámetros, los hiperparámetros se fijan antes del entrenamiento y la validación es el juez que nos dice cuáles son los mejores.

Diferencias Críticas y Flujo de Datos

🏋️ Entrenamiento (Train)

Ajusta los parámetros internos (pesos, reglas). Es donde el modelo "aprende" realmente.

🧪 Validación (Val)

Evalúa configuraciones externas (hiperparámetros). Influye de forma indirecta en el modelo al actuar como guía de optimización.

🏁 Test (Prueba Final)

Evaluación final e imparcial. No influye en el proceso de construcción y solo se usa una vez al final.

Proporciones y Estrategia

Reparto Estándar

Es habitual reservar entre un 15% y un 20% para validación. La clave es que sea lo suficientemente representativo sin degradar excesivamente el tamaño del entrenamiento.

Riesgo de Sobreajuste

Si iteramos demasiadas veces ajustando el modelo basándonos en la validación, el modelo puede terminar "memorizando" accidentalmente este conjunto, introduciendo sesgo.

Flujo de Trabajo Típico

1

Iteración de Versiones

Se entrenan múltiples modelos con diferentes configuraciones (hiperparámetros).

2

Selección en Validación

Se elige la configuración que obtiene mejores resultados en el conjunto de validación.

3

Certificación en Test

Una vez fijado el "ganador", se evalúa por primera y única vez en el test para obtener la métrica real.

Atención

El conjunto de validación se utiliza para AJUSTAR (tuning) el modelo, mientras que el conjunto de test se utiliza para EVALUAR su rendimiento final. El error de validación nunca es imparcial, es una guía de desarrollo.
