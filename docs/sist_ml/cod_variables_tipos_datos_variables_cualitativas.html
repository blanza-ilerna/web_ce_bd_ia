Variables Cualitativas

Atributos y categorías: la información que no vive en los números.

¿Qué son las Variables Cualitativas?

A diferencia de las cuantitativas, estas variables representan cualidades o atributos que clasifican a los elementos en grupos o clases. No indican una cantidad medible, sino una pertenencia.

El número como etiqueta

Aunque operativamente les asignemos códigos numéricos (como 0 para "Hombre" y 1 para "Mujer"), estos números son simples etiquetas. Carecen de propiedades matemáticas: sumar o promediar estas etiquetas carece por completo de sentido lógico.

🏷️

Clasificación de las Variables Categóricas

Es vital diferenciar si existe una jerarquía interna entre las categorías para elegir la codificación adecuada:

1. Variables Nominales

Las categorías solo sirven para nombrar o identificar grupos. No existe ningún orden lógico o secuencia entre ellas.

  • 🎨 Color (rojo, verde, azul)
  • 🏭 Tipo de industria
  • ⚧️ Género
  • 🆔 ID de producto

El orden de presentación es irrelevante.

2. Variables Ordinales

En este caso, sí existe un orden o jerarquía claro. Podemos decir que una categoría es superior o precede a la otra.

  • 🎓 Nivel educativo (ESO, Bachillerato, Grado)
  • 📈 Nivel socioeconómico (Bajo, Medio, Alto)
  • ⭐ Satisfacción (Mala, Neutral, Buena)
  • 🏆 Calificación crediticia (A, B, C)

Hay orden, pero no podemos medir la "distancia" exacta entre niveles.

💡 Importancia de la Naturaleza del Dato

Saber si nuestra variable cualitativa es nominal u ordinal determinará cómo la traduciremos a números en las fases posteriores. Como vimos anteriormente, ignorar el orden en una variable ordinal o inventarlo en una nominal son errores que degradan la capacidad predictiva de nuestro modelo.
