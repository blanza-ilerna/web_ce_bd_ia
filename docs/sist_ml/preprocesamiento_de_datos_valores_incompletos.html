Eliminación de Registros Incompletos

La solución más directa para el tratamiento de datos faltantes.

¿En qué consiste este método?

La eliminación de registros incompletos consiste en borrar aquellas observaciones que contienen valores ausentes en una o más variables relevantes. Se trata del método más simple y directo para tratar los datos faltantes.

Su uso tiene que estar bien pensado, ya que es un método que nos lleva inevitablemente a la destrucción y pérdida de información relevante para el modelo.

¿Cuándo es apropiado usarlo?

Este enfoque es adecuado únicamente bajo ciertas condiciones estrictas:

Proporción Reducida

Cuando la proporción de datos ausentes es muy baja (por ejemplo, inferior al 5–10% del total).

Muestreo MCAR

Cuando los valores perdidos son completamente al azar (MCAR) y no dependen de otras variables.

Riesgos de la Eliminación Masiva

Si las ausencias están asociadas con el valor de otras variables o con el fenómeno que se estudia (MAR o MNAR), eliminar registros puede provocar una distorsión grave.

  • Distorsión de resultados: Se altera la distribución estadística original.
  • Pérdida de representatividad: El modelo puede ignorar patrones de grupos específicos.
  • Baja generalización: El sistema fallará al enfrentarse a datos reales con ausencias similares.

Por tanto, este método debe considerarse una solución práctica, pero no necesariamente óptima, especialmente si la pérdida de información es sustancial.

Implementación con Python y Pandas

En entornos de producción, solemos utilizar la librería pandas para identificar y descartar rápidamente estas filas:

limpieza_nulos.py
# Identificamos el volumen de nulos por columna
print("Valores ausentes:")
print(df_masked.isna().sum())

# Eliminamos los registros incompletos (todas las filas con NaN)
df_clean = df_masked.dropna()

print(f"Filas resultantes: {len(df_clean)}")

🏁 Conclusión Directa

Borrar es fácil, pero caro en términos de información. Antes de aplicar un .dropna(), asegúrate de que el "hueco" que dejas no es más valioso que la simplicidad que ganas.
