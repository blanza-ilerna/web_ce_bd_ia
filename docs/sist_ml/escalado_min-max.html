Escalado Min-Max

Normalizando datos en un rango acotado de [0, 1].

¿Qué es el MinMaxScaler?

El Escalado Min-Max (o normalización) es un método de preprocesamiento que transforma las características escalando cada valor a un rango específico, comúnmente entre 0 y 1.

Este proceso es esencial para algoritmos que no manejan bien diferencias de magnitud grandes, permitiendo que todas las variables tengan el mismo peso relativo inicial.

X' =
X - Xmin Xmax - Xmin

Preservación Visual

Una de sus grandes ventajas es que preserva la forma original de la distribución. No distorsiona la relación relativa entre los datos.

Sensibilidad a Outliers

Es muy sensible a valores extremos. Un solo outlier puede "comprimir" el resto de los datos en un rango minúsculo, ya que los valores máximo y mínimo definen el nuevo espacio.

Parámetros principales

Aunque a menudo lo usamos con sus valores por defecto, el MinMaxScaler permite cierta personalización estratégica:

Parámetro Defecto Descripción
feature_range (0, 1) Define el rango deseado de los datos transformados. Útil si necesitas, por ejemplo, (-1, 1).
copy True Si es False, intenta realizar la transformación in-place para ahorrar memoria.
clip False Si es True, los valores que se salgan del rango en nuevos datos serán forzados a los límites (0 o 1).

Ejemplo Práctico Avanzado

En este escenario, vamos a escalar un dataset completo de propiedades inmobiliarias, ajustando el rango para que los valores oscilen entre 1 y 10 (útil para sistemas basados en puntuaciones).

advanced_scaling.py 
from sklearn.preprocessing import MinMaxScaler

# Configuramos un rango personalizado del 1 al 10
scaler = MinMaxScaler(feature_range=(1, 10))

# Seleccionamos las columnas numéricas que deseamos normalizar
cols_to_scale = ['precio', 'habitaciones', 'metros_cuadrados']

# Aplicamos el escalado a todo el bloque a la vez
df[cols_to_scale] = scaler.fit_transform(df[cols_to_scale])

# Ahora 'precio' y 'metros_cuadrados' son comparables numéricamente
print(df[cols_to_scale].head())

Tip Pro: Al aplicar el escalado a varias columnas a la vez, MinMaxScaler calcula los máximos y mínimos de manera independiente para cada una de ellas.
