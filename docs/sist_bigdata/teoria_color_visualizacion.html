  • Protanopia: 2.5% poblaci√≥n (rojo-verde, severa)
  • Tritanopia: 0.001% poblaci√≥n (azul-amarillo, rara)
  • Total global: ~300 millones de personas con CVD

Tipos de Daltonismo y Su Impacto en Visualizaci√≥n

Deuteranopia (5% poblaci√≥n)

Afecta: Cono verde (dificultad rojo-verde)

Confunde:

  • Verde ‚Üî Marr√≥n
  • Rojo ‚Üî Verde oscuro
  • Azul ‚Üî Morado

Soluci√≥n: Usar azul-naranja o agregar patrones/formas

Protanopia (2.5% poblaci√≥n)

Afecta: Cono rojo (rojo-verde severo)

Confunde:

  • Rojo ‚Üî Negro
  • Verde ‚Üî Amarillo p√°lido
  • Naranja ‚Üî Verde

Soluci√≥n: Viridis o Cividis (ning√∫n rojo)

Tritanopia (0.001% poblaci√≥n)

Afecta: Cono azul (azul-amarillo, muy rara)

Confunde:

  • Azul ‚Üî Verde
  • Amarillo ‚Üî Rosa/Violeta

Soluci√≥n: Rojo-negro es seguro

Acromatopsia (<0.0001%)

Afecta: Todos los conos (visi√≥n monocrom√°tica)

Ve: Solo escala de grises

Soluci√≥n: Usar patrones, texturas, formas diferentes adem√°s de color

Est√°ndares de Accesibilidad: WCAG 2.1 AA

Las Web Content Accessibility Guidelines (WCAG) 2.1 nivel AA son el est√°ndar legal en muchos pa√≠ses (incluyendo UE, USA, Canad√°). Requisitos clave para visualizaciones:

Contraste de Color (WCAG 1.4.3)

  • Texto normal: Ratio de contraste m√≠nimo 4.5:1 con fondo
  • Texto grande (>18pt): Ratio m√≠nimo 3:1
  • Componentes UI: Contraste m√≠nimo 3:1 para barras, l√≠neas

F√≥rmula de Contraste:

Contrast Ratio = (L1 + 0.05) / (L2 + 0.05)

donde:
- L1 = luminancia relativa del color m√°s claro
- L2 = luminancia relativa del color m√°s oscuro
- Luminancia = 0.2126*R + 0.7152*G + 0.0722*B (RGB en escala 0-1)

Ejemplos de Ratios:

  • Negro (#000000) sobre blanco (#FFFFFF): 21:1 (m√°ximo posible)
  • Azul oscuro (#2E86AB) sobre blanco: 4.6:1 ‚úÖ WCAG AA
  • Gris claro (#CCCCCC) sobre blanco: 1.6:1 ‚ùå Falla WCAG

Herramienta: WebAIM Contrast Checker

Estrategias de Dise√±o Accesible

Estrategia 1: Redundancia de Codificaci√≥n

No usar SOLO color para transmitir informaci√≥n. Combinar con:

  • Formas: C√≠rculo, cuadrado, tri√°ngulo en scatter plots
  • Patrones: Rayado, puntos, s√≥lido en barras
  • Etiquetas: Texto directo sobre elementos
  • Tama√±o: Grosor de l√≠nea variable

Estrategia 2: Paletas CVD-Safe

Usar solo paletas cient√≠ficamente validadas:

  • ‚úÖ Viridis, Cividis (deuteranopia-safe)
  • ‚úÖ ColorBrewer con filtro "colorblind safe"
  • ‚úÖ Coolwarm (mejor que RdBu)
  • ‚ùå Jet, Rainbow, RdYlGn est√°ndar

Estrategia 3: Testing con Simuladores

Herramientas para simular CVD:

  • Coblis: color-blindness.com/coblis-color-blindness-simulator/
  • Chrome Extension: "Colorblindly"
  • Python: colorspacious.simulate_cvd()

Estrategia 4: Alto Contraste por Defecto

Dise√±ar con contraste alto desde inicio:

  • Fondo blanco (#FFFFFF) o gris muy claro (#F8F8F8)
  • Texto negro (#000000) o gris oscuro (#333333)
  • Evitar fondos con gradientes o im√°genes

def simulate_deuteranopia(rgb):
    """
    Simula deuteranopia (daltonismo rojo-verde) aplicando matriz de transformaci√≥n.

    Basado en: Brettel, H., Vi√©not, F., & Mollon, J. D. (1997).
    "Computerized simulation of color appearance for dichromats"
    """
    # Matriz de transformaci√≥n para deuteranopia
    transform_matrix = np.array([
        [0.625, 0.375, 0.0],  # R
        [0.7, 0.3, 0.0],      # G
        [0.0, 0.3, 0.7]       # B
    ])

    # Aplicar transformaci√≥n
    rgb_deuter = np.dot(rgb, transform_matrix.T)
    return np.clip(rgb_deuter, 0, 1)

def simulate_protanopia(rgb):
    """Simula protanopia (ceguera al rojo)"""
    transform_matrix = np.array([
        [0.56667, 0.43333, 0.0],
        [0.55833, 0.44167, 0.0],
        [0.0, 0.24167, 0.75833]
    ])
    rgb_protan = np.dot(rgb, transform_matrix.T)
    return np.clip(rgb_protan, 0, 1)

def palette_distinguishability(palette):
    """
    Calcula distinguibilidad: suma de distancias euclidianas entre colores consecutivos.
    Mayor valor = m√°s distinguible.
    """
    distances = []
    for i in range(len(palette) - 1):
        dist = np.linalg.norm(palette[i] - palette[i+1])
        distances.append(dist)
    return np.sum(distances)
