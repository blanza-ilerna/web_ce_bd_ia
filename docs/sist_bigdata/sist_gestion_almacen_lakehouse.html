El Mejor de los Dos Mundos

El concepto de Lakehouse surge para resolver la brecha entre el Data Lake y el Data Warehouse. Combina la escalabilidad y bajo coste del primero con la estructura, integridad y gobernanza del segundo.

Esta arquitectura se basa en el uso de formatos de tabla transaccional que incorporan las propiedades ACID (Atomicidad, Consistencia, Aislamiento y Durabilidad) sobre sistemas de almacenamiento distribuido como S3 o ADLS.
