B. DE McCULLOCH-PITTS AL MLP

McCulloch-Pitts (1943)

Entradas binarias (0/1). Pesos fijos (excitatorios + / inhibitorios −). Umbral fijo θ.

Salida = 1 si Σ(wᵢ·xᵢ) ≥ θ, si no → 0

✗ Sin aprendizaje. ✗ Solo binario. ✗ Pesos manuales.

MIASAA
Perceptrón de Rosenblatt (1957)

Añade pesos ajustables. Aprende mediante la regla delta: si falla, ajusta los pesos.

w ← w + α·(y_real − y_pred)·x

✓ Aprende. ✗ Solo clases linealmente separables. ✗ No resuelve XOR.

MIASAAPIA
MLP — Perceptrón Multicapa

Varias capas ocultas + funciones de activación no lineales + Backpropagation. Resuelve problemas no lineales (XOR, reconocimiento de patrones…).

y = f(W·x + b) con múltiples capas

✓ No-lineal. ✓ Aprende. ✓ Generalizable. Base del Deep Learning.

C. DEFINICIÓN DE DEEP LEARNING

¿Qué es el Deep Learning?

Subcampo del Machine Learning basado en Redes Neuronales Artificiales con múltiples capas ocultas ("profundas"). Aprende representaciones jerárquicas de los datos de forma automática: cada capa extrae características más abstractas que la anterior.

Analogía: como el cerebro visual humano. Capa 1 detecta bordes, capa 2 formas, capa 3 objetos, capa N conceptos abstractos.

ML → Aprendizaje de features manuales  |  DL → Aprendizaje automático de representaciones
MIASAAPIASBD
¿Cuándo usar Deep Learning?
  • Datos en gran volumen (>100K muestras).
  • Datos no estructurados: imágenes, audio, texto.
  • Relaciones muy complejas y no lineales.
  • Features difíciles de diseñar manualmente.
  • Potencia de cómputo disponible (GPU/TPU).
¿Cuándo NO usar Deep Learning?
  • Pocos datos etiquetados (<1K).
  • Se necesita interpretabilidad del modelo.
  • Datos tabulares simples → Random Forest/XGBoost es mejor.
  • Recursos limitados (CPU, memoria).
  • Tiempo de desarrollo corto.

E. ARQUITECTURA DE UNA RED — CAPAS

Capa de Entrada (Input Layer)

Recibe los datos brutos. No realiza cálculos. Número de neuronas = número de features del problema.

Ej: 784 neuronas para imagen 28×28px (MNIST).

Capas Ocultas (Hidden Layers)

Extraen representaciones. Cada capa aprende características más abstractas. Más capas = más profundo = más capacidad de aprender patrones complejos.

El "corazón" del aprendizaje profundo.

Capa de Salida (Output Layer)

Produce la predicción final. Neuronas = número de clases (clasificación) o 1 (regresión).

Función de activación: Softmax (multiclase), Sigmoid (binaria) o Lineal (regresión).

Backpropagation — ¿Cómo aprende la red?
  1. Forward pass: Los datos fluyen de entrada a salida. Cada neurona calcula z = Σwx+b → f(z).
  2. Cálculo del error: La función de pérdida (Loss) compara la salida real con la predicha.
  3. Backward pass: El gradiente del error se propaga hacia atrás capa a capa (regla de la cadena).
  4. Actualización de pesos: w ← w − α·∂L/∂w   (α = learning rate).
  5. Repetir durante N épocas hasta convergencia.

F. FUNCIONES DE ACTIVACIÓN

FunciónFórmulaRangoUso típicoVentaja / ProblemaEn el temario
Escalón (Step) 1 si z≥0, 0 si z<0 {0, 1} McCulloch-Pitts, Perceptrón clásico. Interpretable. ✗ No diferenciable → no backprop. MIA SAA
Sigmoide (σ) σ(z) = 1/(1+e⁻ᶻ) (0, 1) Clasificación binaria (capa salida). Regresión logística. Salida = probabilidad. ✗ Vanishing gradient en capas profundas. MIA SAA PIA
Tanh tanh(z) = (eᶻ−e⁻ᶻ)/(eᶻ+e⁻ᶻ) (−1, 1) Capas ocultas de RNNs. Señales centradas en 0. Media cero, mejor que sigmoide. ✗ También vanishing gradient. MIA SAA
ReLU max(0, z) [0, +∞) Capas ocultas. Estándar en CNN y MLP. Rápida, simple, no vanishing. ✗ "Neurona muerta": si z<0 siempre, gradiente=0. MIA SAA PIA
Leaky ReLU z si z≥0, 0.01z si z<0 (−∞, +∞) Capas ocultas. Alternativa a ReLU cuando hay neuronas muertas. Evita neuronas muertas. Pendiente pequeña para z<0. MIA SAA
Softmax e^zᵢ / Σe^zⱼ (0,1) suma=1 Capa de salida para clasificación multiclase. Convierte logits en distribución de probabilidad. Una clase por neurona. MIA SAA PIA
Lineal (Identidad) f(z) = z (−∞, +∞) Capa de salida para regresión. Predice valores continuos. Sin limitación de rango. ✗ Sin no-linealidad. SAA PIA
GELU Aprox. 0.5z(1+tanh(√(2/π)(z+0.044z³))) (−0.17, +∞) Transformers (GPT, BERT). Capas ocultas de LLMs. Mejor que ReLU en modelos de atención. Suave en z<0. MIA SAA

G. TIPOS DE REDES NEURONALES

RedArquitectura claveDatos idealesAplicacionesEn el temario
MLP (Perceptrón Multicapa) Capas densas completamente conectadas. Forward feed. Backprop. Datos tabulares, vectores de features. Clasificación, regresión, reconocimiento de patrones simples. MIA SAA PIA
CNN (Red Neuronal Convolucional) Capas convolucionales (filtros) + Pooling + Capas densas. Detecta patrones locales con pesos compartidos. Imágenes, vídeo, señales 2D/1D. Visión por computadora, clasificación de imágenes, detección de objetos, reconocimiento facial. MIA SAA PIA SBD
RNN (Red Neuronal Recurrente) Conexiones recurrentes. Estado oculto (memoria). Procesa secuencias paso a paso. Secuencias temporales, texto, audio, series temporales. Traducción automática, predicción de texto, análisis de sentimiento, series financieras. MIA SAA PIA
LSTM / GRU RNN con compuertas (forget, input, output). Controla qué recordar y qué olvidar. Resuelve vanishing gradient en RNN. Secuencias largas. Texto con dependencias a distancia. NLP, traducción, generación de texto, reconocimiento de voz, predicción de series largas. MIA SAA PIA
Transformer Mecanismo de atención (Self-Attention). Sin recurrencia → paralelizable. Codificador + Decodificador. Texto, imágenes (ViT), audio. LLMs (GPT, BERT, LLaMA), traducción, resumen, generación de código, visión (ViT). MIA SAA PIA
Autoencoder Encoder (comprime) + Bottleneck + Decoder (reconstruye). Aprende representación comprimida sin etiquetas. Datos sin etiquetar. Imágenes, texto. Reducción dimensionalidad, detección de anomalías, compresión, generación de datos. MIA SAA
GAN (Red Generativa Adversarial) Generador (crea muestras) vs Discriminador (distingue real/falso). Entrenamiento adversarial. Imágenes, audio, texto. Generación de imágenes realistas, deepfakes, síntesis de datos, arte IA. MIA SAA
