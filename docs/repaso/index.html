  • 💻 Cheatsheet: Programación, Librerías y Software - Software empresarial, lenguajes, frameworks de modelado, compilación e IA sintáctica.

  • 🚀 Cheatsheet: SAA Tema 1. Inteligencia Fuerte y Débil - Niveles ANI, AGI, ASI, barrera evolutiva de Nick Bostrom y el dilema del Alineamiento Humano.

  • 🕶️ Cheatsheet: SAA Tema 4. Aprendizaje NO Supervisado - Clustering masivo (K-Means/DBSCAN), purga dimensional (PCA) y reglas de recomendación (Apriori).
  • 🧹 Cheatsheet: SAA Tema 5. Preprocesamiento y Limpieza - Filosofía GIGO, imputación KNN de Nulos, One-Hot Encoding categórico y escaladores (Standard/Robust).
  • 🎯 Cheatsheet: SAA Tema 6. Optimización y Tuning - Selección de variables (Filter/Wrapper), Hiperparámetros, Grid Search, Random Search y Optimización Bayesiana.
  • 🧠 Cheatsheet: SAA Tema 7. Deep Learning - Backpropagation, CNN (Visión), RNN/LSTM (Tiempo), GANs (Generativas) y arquitectura Transformer (LLM).
  • 📐 Cheatsheet: SAA Nuevo Tema 6. Valoración de Calidad - Capacidad de Generalización, pelígro de Data Leakage, técnicas Early Stopping y A/B Testing MLOps.
