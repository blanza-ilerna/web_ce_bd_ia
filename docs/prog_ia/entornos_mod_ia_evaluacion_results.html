Evaluaci√≥n de Resultados

Midiendo el √âxito de los Modelos de IA

La Importancia de Medir

Entrenar un modelo es solo la mitad del trabajo. Una vez finalizado el entrenamiento, es esencial medir su rendimiento rigurosamente para garantizar que cumple con los objetivos del negocio y generaliza correctamente ante nuevos datos.

No existe una m√©trica universal. La elecci√≥n depende totalmente del tipo de problema: ¬øestamos clasificando correos spam, prediciendo el precio de una casa o agrupando clientes por comportamiento?

M√©tricas para Clasificaci√≥n

Utilizadas cuando la variable objetivo es una categor√≠a (ej. S√≠/No, Gato/Perro).

Precisi√≥n (Precision)

De todos los casos que predije como positivos, ¬øcu√°ntos eran realmente positivos?

Recall (Sensibilidad)

De todos los casos positivos reales, ¬øcu√°ntos fue capaz de detectar el modelo?

F1-Score

La media arm√≥nica entre Precisi√≥n y Recall. Ideal cuando hay clases desbalanceadas.

Exactitud (Accuracy)

El porcentaje total de aciertos. Puede ser enga√±osa si el dataset est√° desbalanceado.

Ejemplo Pr√°ctico con Scikit-learn

from sklearn.metrics import classification_report, confusion_matrix

# Valores reales vs Predicciones del modelo
y_real    = [0, 1, 1, 0, 1, 1, 0, 0, 1, 0]
y_predich = [0, 1, 0, 0, 1, 1, 0, 1, 1, 0]

print("Informe de Clasificaci√≥n:")
print(classification_report(y_real, y_predich, target_names=['Clase 0', 'Clase 1']))

# Matriz de Confusi√≥n
# [[TN, FP],
#  [FN, TP]]
print("Matriz de Confusi√≥n:")
print(confusion_matrix(y_real, y_predich))

M√©tricas para Regresi√≥n

Utilizadas cuando se predice un valor num√©rico continuo (ej. temperatura, precio).

Error Cuadr√°tico Medio (MSE) y R¬≤

  • MSE (Mean Squared Error): Penaliza fuertemente los errores grandes al elevar la diferencia al cuadrado.
  • MAE (Mean Absolute Error): Promedio de la diferencia absoluta. M√°s interpretable.
  • R¬≤ (Coeficiente de Determinaci√≥n): Indica qu√© porcentaje de la varianza de los datos explica el modelo (1.0 es perfecto).
from sklearn.metrics import mean_squared_error, r2_score
import numpy as np

# Precios reales vs Predichos (en miles de euros)
precios_reales = np.array([100, 150, 200, 250])
precios_predic = np.array([110, 140, 210, 245])

mse = mean_squared_error(precios_reales, precios_predic)
r2 = r2_score(precios_reales, precios_predic)

print(f"MSE: {mse:.2f}")  # Salida: 112.50
print(f"R¬≤ Score: {r2:.4f}") # Salida: 0.96 (Muy buen ajuste)

M√©tricas para Clustering (No Supervisado)

Al no tener etiquetas "correctas", medir la calidad de los grupos es m√°s complejo.

√çndice de Silueta

Mide cu√°n similar es un objeto a su propio grupo (cohesi√≥n) comparado con otros grupos (separaci√≥n). Var√≠a de -1 a 1.

Entrop√≠a / Inercia

Mide la "pureza" de los cl√∫steres o la suma de las distancias al cuadrado de las muestras a su centro de cl√∫ster m√°s cercano.

Herramientas de Visualizaci√≥n Avanzada

TensorBoard

Herramienta de visualizaci√≥n de Google incluida con TensorFlow. Permite monitorear experimentos en tiempo real, ver curvas de p√©rdida (loss) y precisi√≥n (accuracy) durante el entrenamiento, y visualizar grafos de modelos complejos.

Uso b√°sico mediante Callback:

          epochs=5, 
          callbacks=[tensorboard_callback])

# Para ver los resultados, ejecutar en terminal:
# tensorboard --logdir logs/fit
