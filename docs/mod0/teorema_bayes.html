¬øQu√© es el Teorema de Bayes?

El Teorema de Bayes es una de las herramientas matem√°ticas m√°s poderosas y fundamentales en el campo de la probabilidad y la estad√≠stica. Permite actualizar nuestras creencias a medida que obtenemos nueva evidencia.

La F√≥rmula del Teorema de Bayes

P(A|B) = [P(B|A) √ó P(A)] / P(B)

La probabilidad de A dado B es igual a la probabilidad de B dado A multiplicada por la probabilidad de A, todo dividido por la probabilidad de B

Componentes de la F√≥rmula

P(A|B) - Posterior

Lo que queremos calcular: nuestra creencia actualizada.

P(B|A) - Likelihood

Probabilidad de observar la evidencia si A es verdadero.

P(A) - Prior

Nuestra creencia inicial antes de ver la evidencia.

P(B) - Evidencia

Probabilidad total de observar B. Normaliza el resultado.

Ejemplo M√©dico Detallado

Supongamos una enfermedad rara que afecta al 0.1% de la poblaci√≥n (1 de cada 1,000 personas). Existe un test con:

Sensibilidad: 99%

Si est√°s enfermo, el test ser√° positivo el 99% de las veces.

P(Test+|Enfermo) = 0.99

Especificidad: 99%

Si est√°s sano, el test ser√° negativo el 99% de las veces.

Pregunta Clave

Si el test da POSITIVO, ¬øcu√°l es la probabilidad de que realmente est√©s enfermo?

La mayor√≠a responde intuitivamente "99%" (la sensibilidad del test).

Aplicando el Teorema de Bayes

Paso 1: Identificar los valores conocidos

  • P(Enfermo) = 0.001 (prevalencia: 0.1%)
  • P(Test+ | Enfermo) = 0.99 (sensibilidad)
  • P(Test+ | Sano) = 0.01 (falso positivo: 1 - especificidad)

Paso 2: Calcular P(Test+)

P(Test+) = P(Test+|Enfermo) √ó P(Enfermo) + P(Test+|Sano) √ó P(Sano)
P(Test+) = (0.99 √ó 0.001) + (0.01 √ó 0.999)
P(Test+) = 0.00099 + 0.00999 = 0.01098

Paso 3: Aplicar la f√≥rmula de Bayes

P(Enfermo|Test+) = [P(Test+|Enfermo) √ó P(Enfermo)] / P(Test+)
P(Enfermo|Test+) = (0.99 √ó 0.001) / 0.01098
P(Enfermo|Test+) = 0.00099 / 0.01098 ‚âà 0.0902

RESULTADO FINAL

P(Enfermo|Test+) ‚âà 9.02%

Aunque el test tiene 99% de precisi√≥n, si da positivo solo hay ~9% de probabilidad de estar realmente enfermo. ¬°La intuici√≥n humana falla aqu√≠!

¬øPor qu√© este resultado sorprendente?

La clave est√° en la prevalencia baja de la enfermedad (0.1%). De cada 1,000 personas:

  • 1 est√° enferma, 999 est√°n sanas
  • El test detectar√° correctamente: 1 √ó 0.99 ‚âà 1 positivo verdadero
  • Pero dar√° falsos positivos: 999 √ó 0.01 ‚âà 10 positivos falsos
  • Total de positivos: 11 personas (1 verdadero + 10 falsos)
  • Probabilidad real: 1/11 ‚âà 9%

Implementaci√≥n en Python

Funci√≥n B√°sica del Teorema de Bayes

def teorema_bayes(prior, likelihood, evidencia):
    """Calcula P(A|B) usando el Teorema de Bayes"""
    posterior = (likelihood * prior) / evidencia
    return posterior

# Ejemplo: Diagn√≥stico m√©dico
P_enfermo = 0.001
P_test_pos_dado_enfermo = 0.99
P_test_pos_dado_sano = 0.01
P_sano = 1 - P_enfermo

# Calcular P(Test+)
P_test_positivo = (P_test_pos_dado_enfermo * P_enfermo +
                   P_test_pos_dado_sano * P_sano)

# Aplicar Teorema de Bayes
resultado = teorema_bayes(
    prior=P_enfermo,
    likelihood=P_test_pos_dado_enfermo,
    evidencia=P_test_positivo
)

print(f"Probabilidad: {resultado:.4f} = {resultado*100:.2f}%")
# Salida: Probabilidad: 0.0902 = 9.02%

Naive Bayes con scikit-learn

from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

# Dataset de emails

    "Gana dinero r√°pido! Haz clic aqu√≠",      # SPAM
    "Oferta limitada! Compra ahora",          # SPAM
    "Reuni√≥n ma√±ana a las 10",                # NO SPAM
    "¬øNos vemos para cenar el viernes?",      # NO SPAM
]
etiquetas = [1, 1, 0, 0]  # 1=spam, 0=no spam

# Convertir texto a vectores (Bag of Words)
vectorizador = CountVectorizer()
X = vectorizador.fit_transform(mensajes)

# Entrenar clasificador

clasificador.fit(X, etiquetas)

# Predecir nuevos mensajes
nuevos = ["Oferta incre√≠ble compra ya", "¬øConfirmamos reuni√≥n?"]

predicciones = clasificador.predict(X_nuevos)

for i, msg in enumerate(nuevos):
    clase = "SPAM" if predicciones[i] == 1 else "NO SPAM"
    print(f"{msg} ‚Üí {clase}")

# Salida:
# Oferta incre√≠ble compra ya ‚Üí SPAM
# ¬øConfirmamos reuni√≥n? ‚Üí NO SPAM
